## Installation

To get started with the project, follow these steps:

1. Clone the code:
   ```bash
   $ git clone https://github.com/JustineUgo/task_todo.git
   $ cd task_todo
    ```

2. Run the following commands:
   ```bash
   $ flutter pub get
    ```
3. Run app:
   ```bash
   $ flutter run
    ```
